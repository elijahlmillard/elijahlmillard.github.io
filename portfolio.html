PROJECTS

KCPS CONTENT HUB

A SharePoint-based knowledge base developed for Kansas City Public Schools to reduce internal support tickets by centralizing employee resources. Built custom web parts using TypeScript, React, and the SharePoint Framework (SPFx) for direct deployment to SharePoint pages.

COBOL PROGRAMS — PUB400

Currently developing and running COBOL programs live on PUB400 under library ELIJAHM1. Focused on understanding the fundamentals of legacy enterprise programming — data structures, file handling, and the architecture that still powers a significant portion of the world's critical infrastructure.

ARDUBOY DEVELOPMENT

Programming games and applications for the Arduboy — a credit card sized open source game system built on Arduino architecture. Working in C++ to build constrained, efficient programs within tight memory and processing limitations. A study in doing more with less.

THIS PORTFOLIO

Built from scratch in vanilla HTML, CSS, and JavaScript. No frameworks, no libraries, no build tools. Features an animated 1895-era silent film intro sequence, canvas-based film grain running at 24fps, and a 35mm film strip navigation system. Hosted on GitHub Pages.

BUSINESS

The i.c.stars program operates as a consulting environment. Cohort members form consulting companies, respond to RFPs, and deliver real solutions to real clients under professional conditions.

The KCPS Content Hub project presented a genuine business problem — employees across the district lacked a centralized knowledge base, resulting in high volumes of internal support tickets. The solution required understanding the client's infrastructure, their existing SharePoint environment, and the communication gaps driving the problem.

Company Engagement Days with partners including Accenture and Salesforce provided direct exposure to enterprise consulting culture, professional communication standards, and the expectations of technology employers in the Kansas City market.
